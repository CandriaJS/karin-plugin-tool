## 介绍 📝
`柠糖工具` 是一个 `Yunzai-Bot` | `Karin` 的扩展插件，提供工具类功能， 如网页截图等

如有问题请提交 `issue` 或加入 Q 群: `272040396` 📬

## 安装与更新 🔧

在`Karin`目录执行以下命令

```bash
pnpm add @candriajs/karin-plugin-tool@latest -w
```
